Versionamento de Código com Git e Github
Criar pontos na história de um projeto conforme vai desenvolvendo

Árvore com um único Branch(galho)
  • git init >> Informar que quer controlar a pasta pelo git master (caule da árvore)
  • git status >> Informa status dos arquivos que já estão sendo controlados ou não pelo git
  • git add index.html >> Adicionar arquivo ao ponto da história
    • git add *.html >> Adiciona todos arquivos html
    • git add . >> Adiciona todos os arquivos
  • git commit -m "Adicionado index.html" >> Primeiro commit informando alteração do Código
  • git log >> Exibe os commits que já foram realizados(pontos na história). Tele q para fechar
    • git log -2 >> Exibe os dois últimos commits realizados
  • git diff >> Mostra as alterações realizadas
  • Para não rastrear arquivos pelo Git você deve criar um arquivo na raiz do projeto com o nome .gitignore e adicione o caminho dos arquivos(nome da pasta) que não quer rastrear
  • Arquivos-Pastas
    • touch .gitignore >> Cria novo arquivo
    • cat .gitignore >> Mostra conteúdo do arquivo
    • rm -rf nomedapasta >> Exclui a pasta
  • git checkout codigodocommit>> Pular para outro ponto da história, você precisa pegar o código do commit ... no log
    • git checkout codigodocommit >> Alternar ponto da história
    • git checkout master >> Voltar ao último commit realizado
Adicionar outro Branch(galho) à Árvore
  • git branch feature/nfe >> Cria um novo branch feature/nfe
  • git branch >> Exibe os branchs existentes
  • git checkout feature/nfe >> pula para a branch
  • git merge feature/nfe >> Adiciona branch criado a árvore principal(Caule) (execute dentro do main/master 'git checkout main')
  • git branch -D feature/nfe >> Exclui a branch
Github
  • Crie um novo repositório, copie a url criada
  • git remote add origin https://github.com/wsawebmaster/git.git >> Define o repositório que será utilizado
  • git remote -v >> Lista as urls remotas
  • git push origin main >> Envia arquivos ao repositório do Github
  • git clone url nomedapasta >> Baixa arquivos do repositório para sua pasta local
  • git pull origin main >> Atualiza informações do github para sua pasta local
Procedimentos para atualizar os dados da pasta local para o Github
  • git add . >> Adicionar todos arquivos ao branch
  • git status >> Confirmar que tudo foi Adicionado
  • git commit -m "Atualização do index" >> Crie o commit para atualizar o branch
  • git push origin main >> Enviar arquivos para o repositório do Github
